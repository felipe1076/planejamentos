# Sistema de Planejamento Escolar (SEMED Altos)

Este é um sistema da web simples, moderno e eficiente para criar planejamentos semanais escolares com exportação direta para **PDF** e **Word**.

## 🚀 Funcionalidades
- **Interface Intuitiva**: Adicione e remova semanas de planejamento com um clique.
- **Calendários Dinâmicos**: Seleção de datas de início e fim individuais para cada semana.
- **Exportação Multiformato**:
  - **PDF**: Em modo Paisagem (A4) focado na tabela de planejamento.
  - **Word (DOC)**: Versão editável da tabela preservando a formatação.
- **100% Offline**: Pode ser executado localmente sem necessidade de servidor.

## 🛠️ Tecnologias
- HTML5 / CSS3 (Design Premium com Inter e Outfit fonts)
- JavaScript (Vanilla)
- [jsPDF](https://github.com/parallax/jsPDF) + [AutoTable](https://github.com/simonbengtsson/jsPDF-AutoTable) (Geração de PDFs)
- [Flatpickr](https://flatpickr.js.org/) (Calendários dinâmicos)

## 📂 Como Usar
1.  Clone este repositório ou baixe os arquivos.
2.  Abra o arquivo `index.html` em qualquer navegador moderno.
3.  Preencha as semanas e gere seus documentos!
